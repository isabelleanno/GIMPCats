Superimposed Image- Cats in Space!

My Process for this image

I used the freehand select tool to select each cat from its background. Then, I used layer masks to hide everything besides the cat. Lastly, I used the blur tool to make the edges of each kitty softer and less blocky, so the edges aren't so sharp. At first, the process was tedious, but I got the hang of it and was able to do it with relative ease.
